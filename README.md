# github-actions-playwright-testiny-example

This repository contains the following examples on how run test automation jobs with GitHub Actions and report results to Testiny: 

* Report Playwright JSON reports (.github/workflows/playwright.yml)
* Report Playwright JUnit XML reports (.github/workflows/playwright-junit.yml)
* Report with Playwright Testiny reporter directly (.github/workflows/playwright-reporter.yml)
* Report from parallel/sharded tests (.github/workflows/playwright-sharded.yml)
